# Axon
This repository presents a bottom-up implementation of a Multilayer Perceptron (MLP) designed for high-throughput training and inference. By bypassing high-level deep learning frameworks, this project focuses on the low-level orchestration of GPGPU (General-Purpose Graphics Processing Units) resources.

Initially, the implementation will be developed using a CPU-based approach to ensure correctness and modularity. However, the architecture is being designed with flexibility in mind, allowing future configuration and extension to GPU execution using custom CUDA kernels.
